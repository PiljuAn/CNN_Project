# CNN 이미지처리 분리수거 분류모델

재활용 쓰레기 분류모델을 개발하여 환경오염 개선에 드는 비용을 절감하고, 정확도를 높이고자 한다  
절차에 따라 투입된 Garbage를 1차 분류하고, Clothes를 상세 분류하는 과정을 거치게 된다
- 팀원 : 김재경, 안필주, 이주복, 이민아
